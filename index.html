Test A: Linked Map and PCP Display

Overview

This visualization represents population data and the associated data uncertainty. Uncertainty represents how reliable the data; the higher the value the less reliable it is for anaylsis. In this visualization, there are four parts: (1) The map on the right, which represents the data spatially.(2) The parallel coordinate plot on the bottom, which has two selectable axes of the uncertainty and population values. The third element is the info label below the map that displays values when the mouse hovers over data on the map. The last element is the legend on the far right, which shows what each color represents.

Task 1

The first task is to select approx. the 50% highest uncertainty values.

Task 2

The second task is to identify if uncertainty appears in a pattern or is generally random across the data.
